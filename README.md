# Kubernetes Ambassador ingress multiple domain cert renewal automated script built in RUST

# Requirements
 * Kubectl
 * kubernetes 
 * Ambassador Edge stack https://www.getambassador.io/
 * cert-manager 

# Install project 
```
$ cargo install --path .
```

# Run Test cases
```
$ cargo test 
```

# Format rust files
```
$ cargo fmt  
```

# Build project 
```
$ cargo build 
```

# Run project
```
$ cargo run 
```
